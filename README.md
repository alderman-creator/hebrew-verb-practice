# Hebrew Verb Practice Site

This folder is publish-ready for GitHub Pages.

## Publish (once authenticated)

1. Create a new public GitHub repo (empty), e.g. `hebrew-verb-practice`.
2. From this folder:

```bash
git init
git add .
git commit -m "Publish Hebrew verb practice app"
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/hebrew-verb-practice.git
git push -u origin main
```

3. In GitHub repo settings, enable Pages from branch `main`, folder `/ (root)`.

Your app URL will be:

`https://<YOUR_USERNAME>.github.io/hebrew-verb-practice/`
